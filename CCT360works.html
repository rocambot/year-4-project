CCT360 Project- Photograph section in My Production

This CCT360 project applied the responsive system from Boostrap 5. In the photograph section here, The grid system of the website is col-xl in the full screen resolution. As the resolution goes smaller. The grid system goes larger. In this case, col-sm-12 is applied when the screen size is less than 768px. And, the pictures are the clickable lightbox gallery.
